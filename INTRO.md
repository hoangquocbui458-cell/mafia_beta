# Mafia Moderator Helper

Короткий входной файл в проект. Здесь только актуальная навигация: что в репозитории, где менять логику, где читать подробности.

## Что в проекте сейчас

- Веб-приложение без сборки: открыл index.html и работаешь.
- Разделенная архитектура: конфиг, роли, движок, UI, интеграционный слой.
- Улучшенный экран настройки ролей:
  - автоматическая рекомендация состава;
  - ручной конструктор с ограничениями;
  - быстрые действия очистить или вернуть рекомендованную раскладку.
- Расширенный игровой цикл:
  - день знакомства;
  - ночные ходы с ограничениями ролей;
  - переголосование при первой ничьей;
  - отдельный экран подтверждения дня без изгнания.
- Встроенный лог партии и экспорт в txt.
- Автотесты движка в autotest.js.

## Карта файлов

```text
mafia_beta/
├── README.md           # Полное руководство по проекту
├── QUICK_START.md      # Практические шаги для частых задач
├── ARCHITECTURE.md     # Архитектурные решения и поток данных
├── INTRO.md            # Этот файл
│
├── index.html          # Разметка экранов и модалок
├── style.css           # Все стили интерфейса
├── config.js           # Роли, порядок ночи, фразы, правила победы
├── roles.js            # Специализированная логика ролей
├── game-engine.js      # Состояние игры и бизнес-правила
├── ui-manager.js       # Рендер и интерфейсное поведение
├── script.js           # Глобальные callbacks и orchestration
├── autotest.js         # Набор автотестов движка
└── script-old.js       # Историческая копия старой реализации
```

## Что читать дальше

1. README.md: если нужно полное описание фич, API и сценариев.
