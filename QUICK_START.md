# 🚀 БЫСТРЫЙ СТАРТ - Новая Архитектура

## ✅ Что изменилось?

Приложение **полностью работает** и **100% совместимо** со старым интерфейсом, но теперь:

- ✨ **Легко добавлять новые роли** - просто отредактируй `config.js` и `roles.js`
- 🎯 **Легко менять фразы** - всё в одном месте (`config.js`)
- 🔧 **Легко добавлять правила** - логика отделена в `roles.js`
- 🧪 **Легко тестировать** - игровой движок в `game-engine.js` независимый
- 📚 **Легко документировать** - каждый файл отвечает за свою область

## 📂 Файлы Проекта

```
config.js          ← ⭐ КОНФИГУРАЦИЯ (редактируй отсюда при добавлении ролей/фраз)
roles.js           ← ⭐ ЛОГИКА РОЛЕЙ (специальная парадигма для каждой роли)
game-engine.js     ← ОСНОВНОЙ ДВИЖОК (игровая логика, не требует редактирования часто)
ui-manager.js      ← ИНТЕРФЕЙС (управление UI, не требует редактирования часто)
script.js          ← ИНИЦИАЛИЗАЦИЯ (callbacks и интеграция, не требует редактирования часто)
index.html         ← УЖ БЕЗ ИЗМЕНЕНИЙ (работает как раньше)
style.css          ← БЕЗ ИЗМЕНЕНИЙ
script-old.js      ← РЕЗЕРВНАЯ КОПИЯ старого кода
```

---

## 🎮 Как Это Работает?

1. **HTML** вызывает функции (типа `addP()`, `go(1)`, `doAction(id)`)
2. **script.js** - это "переводчик", который преобразует вызовы HTML в новую систему
3. **game-engine.js** - управляет состоянием игры и логикой
4. **roles.js** - определяет, как работает каждая роль
5. **config.js** - хранит все настройки и фразы
6. **ui-manager.js** - отрисовывает всё на экран

## ⚠️ Если Что-то Не Работает

### 1️⃣ Ошибка в консоли (F12)?

Проверь:
- Все ли файлы загружены? (в консоли Network tab)
- Правильно ли они подключены в `index.html`?
- Нет ли опечаток в названиях функций?

### 2️⃣ Кнопки не работают?

- Проверь, что функция существует в `script.js`
- Например, `addP()` → это функция в `script.js`

### 3️⃣ Фразы не меняются?

- Отредактируй `config.js` → `PHRASES`
- Сохрани файл
- Обнови страницу (Ctrl+F5, чтобы очистить кеш)

## 📝 Примеры Редактирования

### Добавить новую фразу для Мафии

```javascript
// config.js → PHRASES.rolePrompts.Mafia

'Mafia': [
    "Укажите на того, кто не увидит рассвет. 👺",
    "Моя новая фраза здесь! 🔫",  // ← добавленная
    // ... остальные
]
```

### Изменить название роли

```javascript
// config.js → ROLES

Mafia: {
    displayName: 'Гэнстеры',  // ← изменено с 'Мафия'
    emoji: '👺',
    // ... остальное
}
```

### Изменить количество Мафии по умолчанию

```javascript
// config.js → DEFAULT_ROLE_CONFIG

DEFAULT_ROLE_CONFIG: {
    Mafia: 2,  // ← было 1
    // ... остальное
}
```

## 🎯 Добавление Новой Роли (Краткий Гайд)

### 3 шага:

1. **config.js** - добавить описание роли
   ```javascript
   ROLES: {
       NewRole: {
           displayName: 'Новая роль',
           emoji: '🎭',
           cssClass: 'tag-NewRole',
           alignment: 'good',  // или 'evil' / 'neutral'
           actionType: 'select',  // или 'none'
           isNightRole: true,
           canSkip: true,
           temperament: 'peaceful'  // или 'aggressive'
       }
   }
   ```

2. **config.js** - добавить фразы
   ```javascript
   rolePrompts: {
       'NewRole': ["Фраза 1", "Фраза 2", ...]
   }
   ```

3. **roles.js** - добавить логику
   ```javascript
   NewRole: {
       canSelectTarget(targetIdx, gameState) { ... },
       onNightAction(targetIdx, gameState) { ... },
       getRestrictions() { ... }
   }
   ```

**Готово!** Роль работает! 🎉
